  Delta plan for judge-readiness improvements.

  Human-readable repository map for judges, reviewers, and contributors.

------------------------
- FastAPI application and API routes.
- Synthetic enterprise support tickets, customers, KB articles, and policies.
- Deterministic 7-agent support workflow.
- Structured trajectory span logging.
- Evaluation, failure diagnosis, and optimization engines.
- Mock and OpenAI-compatible LLM provider abstraction.

Primary Local Commands
----------------------
Backend tests:
  cd backend && uv run pytest

Backend server:
  cd backend && uv run uvicorn app.main:app --port 8000

Frontend:
  cd frontend && npm install && npm run dev

Mock benchmark:
  MODE=mock TICKETS=8 REPEATS=3 bash amd/run_agent_benchmark.sh

Winning demo endpoint:
  curl http://localhost:8000/demo/winning-run

AMD vLLM benchmark:
  LLM_BASE_URL=http://localhost:8001/v1 \
  LLM_API_KEY=dummy \
  LLM_MODEL=<model-running-on-vllm> \
  MODE=llm \
  TICKETS=8 \
  REPEATS=3 \
  bash amd/run_agent_benchmark.sh
