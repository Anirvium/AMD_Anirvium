Anirvium AI Repository Structure
================================

Purpose
-------
This repository contains the complete hackathon prototype for Anirvium AI:
Anirvium AI: Sarvagun customer-support execution with SuperTuriya trajectory intelligence.

Primary Local Commands
----------------------
Backend tests:
  cd backend && uv run pytest

Backend server:
  cd backend && uv run uvicorn app.main:app --port 8000

Frontend:
  cd frontend && npm install && npm run dev

Mock benchmark:
  MODE=mock TICKETS=8 REPEATS=3 bash amd/run_agent_benchmark.sh

Winning demo endpoint:
  curl http://localhost:8000/demo/winning-run

AMD vLLM benchmark:
  LLM_BASE_URL=http://localhost:8001/v1 \
  LLM_API_KEY=dummy \
  LLM_MODEL=<model-running-on-vllm> \
  MODE=llm \
  TICKETS=8 \
  REPEATS=3 \
  bash amd/run_agent_benchmark.sh
